  title: 香港八和會館 「粵劇新秀演出系列」公開招募新秀演員 申請須知

  content: >-
    香港八和會館（下稱「本會」）現為康樂及文化事務署轄下油麻地戲院的「場地伙伴」，已踏入第十一年的「粵劇新秀演出系列」（下稱「本計劃」），一直旨在善用油麻地戲院作為本地粵劇的搖籃（至2026年期間，本計劃將會移師高山劇場新翼進行），為本地粵劇業界培育人才。本計劃有幸獲得本地粵劇藝壇前輩及名伶尹飛燕女士、王超群女士、吳仟峰先生、吳美英女士、陳嘉鳴女士、新劍郎先生及龍貫天先生的支持，擔任藝術總監，聯同客席藝術總監羅家英先生，肩負起培訓新秀演員的使命。


    香港八和會館每年於油麻地戲院 / 高山劇場籌辦超過100場粵劇演出及活動，現誠邀有興趣參與培訓及演出的粵劇演員應徵。詳情如下：


    演員應徵資格

    	須為香港居民

    	已接受粵劇表演基本訓練

    	年齡介乎15至40歲\

    	如已在職業班演出，在職業班擔任六柱 / 主要演員不可多於十年


    應徵方法


    1. 有興趣應徵的演員須以個人名義申請，須填妥申請表格，並提供以下資料：
       	近照一張
       	戲裝造型照一張
       	不多於15分鐘的個人粵劇示範(以錄影光碟/ USB 隨身碟/ MPEG 4形式遞交) (屬選擇性項目:如提交有助遴選委員了解應徵者)
    2. 提交方式：只接受以電郵 / 郵寄或親身提交至本會會址之申請，當中需具備上述資料以及已填妥之申請表。(逾時遞交的申請將不獲處理)

    3. 截止日期：2023年3月3日下午5時30分 (本會不接受任何逾期申請。郵寄申請以郵戳為憑，但不會接受因郵資不足 / 郵遞延誤等原因以致逾期提交的申請)

    4. 申請電郵: ymtartists@hkbarwo.com 或 親身提交 / 郵寄地址：九龍油麻地彌敦道493號展望大廈4字樓A座香港八和會館，信封面註明「粵劇新秀演出系列――新秀演員應徵」。

    5. 如欲索取申請須知及表格，或對計劃詳情有任何查詢，歡迎致電香港八和會館 (2384-2939)，或瀏覽「粵劇新秀演出系列」官方網站(www.hkbarwoymt.com)查閱有關資料。

    6. 本會收到應徵者的資料作初步核實後，或會聯絡應徵者提供進一步的資料，以便核實應徵者的資格及安排評選。


    面試安排


    1. 本會稍後將邀請合資格的應徵者進行面試，應徵者在面試之前，將收到兩條考核題目，應徵者可事先自行準備及練習。面試當日，工作人員會在上述兩條題目中選取一條供應徵者即場演繹，進行技藝測試，考生必須在面試時連唱帶做演繹考題。測試項目包括：唱功及演戲。

    2. 遴選委員會主要就應徵者在唱功及演戲兩方面的表現作評估，應徵者需總評分合格外，亦需同時在唱功評分及演戲評分均獲得合格分數，才獲考慮取錄。

    3. 稍後將通知應徵者面試日期、時間及地點。(預計於2023年3月中、後旬舉行)

    4. 遴選委員會將由藝術總監與本計劃督導委員會委員組成。


    申請結考


    1. 本會將個別通知成功獲選的演員，並與其進一步商討演出及培訓等細節安排。

    2. 如應徵者在申請截止後三個月內未有接獲本會的通知，有關申請可視作不成功論。


    基本聘用細則


    1. 新秀演員須與香港八和會館簽訂演出合約。

    2. 新秀演員可獲安排參加粵劇培訓、演出及相關社區推廣及教育活動。本會將參考新秀演員提供的檔期以安排參與演出及相關活動的機會。

    3. 每位新秀演員在各年度擔任主要演員的場數將不多於該年計劃正場場數的70%。新秀演員的薪酬將以實際參與演出的場數及其擔演的角色釐定。

    4. 藝術總監在演期、演員的演出場數、劇目編排、角色分配及選用演員上，具有最終的決策權。

    5. 本計劃以演員培訓為目標，除演出外，新秀演員需參與本會安排的演出前排練和培訓等活動。藝術總監亦會按照新秀演員的個別能力安排培訓項目，如：唱科、基本功、身段、演科、傳統排場及角色行當深造等。

    6. 新秀演員須參與由本會及本計劃資助機構所安排的檢討活動及提交評估報告。


    香港八和會館

    2023年2月13日

    (2023年2月修訂)
