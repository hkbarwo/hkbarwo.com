    香港八和會館(八和)是本港的註冊慈善團體，為粵劇從業員的專業組織。粵劇自2009年列入聯合國教科文組織《人類非物質文化遺產代表作名錄》，香港作為中外文化交流中心，八和一直透過不同計劃和活動與業界內外人士聯動合作，以弘揚劇藝、培育人才為己任，為配合未來發展，現招聘以下職位：  \

    \

    傳訊主任 Communication Officer/助理主任 Assistant Officer (合約) \

    \

    l   協力制定八和的整體傳訊及推廣策略 \

    l   籌劃/執行推廣粵劇藝術的計劃 \

    l   籌劃/執行演出、外展及會員活動的宣傳計劃 \

    l   籌劃/執行經費籌募活動\

    l   協力處理外務事宜 \

    \

    入職要求：\

    l   申請者須為香港永久居民 \

    l   持香港認可大學學位，或具同等學歷 \

    l   操流利中文及英文 \

    l   良好中文及英文書寫能力，熟悉中英文電腦操作\

    l   具相關學歷，及/或有藝團工作經驗，及/或熟悉 Photoshop, Adobe Illustrator, Canva, 影片剪接軟件，可獲優先考慮 \

    l   資歷較淺或應屆畢業生將被考慮為助理主任 (請於應徵信列明)  \

    \

    以上職位(全職)為5天工作，並會在試用期後提供醫療保險。獲聘者需不定時及輪班工作，並於星期六、日及公衆假期的演出當值。有興趣人士請 將求職信連同個人履歷及薪金待遇要求，並在信封上列明申請職位，於2025年5月30日或之前電郵至：hr@hkbarwo.com。  （申請人所提供的資料將予保密及只作招聘有關職位用途） \

    \
     香港八和會館 2025年5月2日
