    **「第六輪資助：保基層就業項目」可能是最後一輪資助 實質資助額須視乎申請數量 有機會少於2萬元**


    粵劇界在過去一年多，隨著表演場館因新型冠狀病毒疫情而三度關閉，生計大受影響。在首度閉館期間，八和成功爭取政府從「防疫抗疫基金」中撥出1,500萬元支援同業。第一、二輪的資助，是直接補貼從業員因演出取消所招致的損失，總共有775位從業員受惠。


    在表演場館第一次重開後(2020年6月19日)，由於觀眾席座位減少，導致票房收入縮減，直接影響營運者(班主)經營的意欲。為保持演出的數量，讓從業員繼續「有工開」，八和在政府的支持下，設立了「第三輪資助：保基層就業項目」。這項資助是希望藉補貼班主營辦演出，以收保障從業員的工作機會之效。獲資助的演出，班主須提供基層從業員的支薪證明，才可獲得資助。


    表演場館分別在2020年7月及12月再次封館，每次場館重開，在政府的同意下，八和均立即推出「保基層就業項目」的資助。直至2021年7月份，我們剛完成「第五輪資助：保基層就業項目」，而三輪的「保基層就業項目」共資助了121場演出，款額超過230萬元。此外，政府於2020年9月及12月分別推出$5,000及$7,500的一筆過資助予從業員，八和繼續協助政府分發資助予粵劇界人士，兩筆資助亦已圓滿完成分發的工作。


    經過了上述多輪的資助，1,500萬的資助，現時約有100萬的餘款。經與政府溝通後，八和現推出「第六輪資助：保基層就業項目」，是項資助的性質，與早前第三至五輪相同：每場演出提供最高$20,000的資助，包括基層從業員的酬金(最高$18,000)及因防疫而產生支出的定額資助$2,000。但由於現時只有約100萬的餘款，如申請資助的演出場數的總申請額，比餘款的數額為高，便會以申請數目和餘款計算每個場次可得的資助上限，即本輪計劃最多可分配給每場演出的資助金上限有機會少於$20,000。與此同時，「第六輪資助：保基層就業項目」也有機會是最後一輪的資助計劃。


    此外，由於「第六輪資助：保基層就業項目」有機會是最後一輪資助，而是次資助的演出期是由2021年5月19日至8月18日，即由政府委託八和分發的「防疫抗疫基金•粵劇界支援計劃」，有機會於8月18日完結。若「第六輪資助：保基層就業項目」是最後一輪資助，即對於8月19日或以後的演出，「防疫抗疫基金•粵劇界支援計劃」將不會再推行任何形式的資助。早前政府康文署已公布，由7月8日起，觀眾席開放率回復至85%，相信隨著觀眾入座率的提升，粵劇演出的營運亦逐步回復正常。有關「第六輪資助：保基層就業項目」的申請資料、資助準則及申請表格，歡迎到https://bit.ly/supportcantonopera下戴。如有任何查詢，請致電2384-2929向本會職員周朗峰先生查詢。


    **「第六輪資助：保基層就業項目」（適用於2021年5月19日至8月18日期間的演出）**

    **申請資料 及 資助準則（申請日期：2021年7月22日至8月4日）**


    **目的**

    1.　疫症過後，康文署表演場館重開時，觀眾的座位數目將會減少，即劇團演出的票房收入亦會縮減，大部份的劇團或會取消演出


    2.　為保持演出的數量，讓從業員「有工開」，確保從業員不會因生計問題而大量流失，令粵劇得以繼續發展及傳承下去，香港八和會館在政府的支持下，設立「第六輪資助：保基層就業項目」，補貼基層從業員的薪酬，誘使經營者繼續舉辦原定的演出


    **資助範圍**

    3.　本計劃資助由康文署開館後，即2021年5月19日至2021年8月18日期間的專業長劇演出，而有關的演出須未曾獲得「防疫抗疫基金」的其他資助


    4.　本計劃不支持以下演出：

    　—　已獲取粵劇發展基金的演出；

    　—　已獲香港藝術發展局資助的演出；

    　—　折子戲、神功戲、演唱會演出；

    　—　凡票房不屬於劇團的演出，如：慈善團體、區議會、民間組織買戲；

    　—　免費派票的演出


    **資助額及計算準則**

    5.　本計劃只資助基層從業員的酬金：　

    　—　每場演出的最高資助額為$18,000

    　—　按比例（請參考第8點）資助日薪$1,200或以下的基層員工


    6.　因防疫而產生支出的定額資助$2,000，劇團只須簡單說明支出的用途，只要是合理的支出便可獲得資助


    7.　如**申請計劃的資助額比可使用的餘款為高**，香港八和會館**將按比例來決定如何分配資助**


    8.　總言之，在資助餘款仍然充足的情況下，每場演出提供可至最高$20,000的資助，當中包括兩部份：甲）基層從業員的酬金（最高$18,000）、乙）因防疫而產生支出的定額資助$2,000


    A)　基層從業員的酬金 （最高$18,000）

    　—　按比例資助酬金$1200以下的基層從業員，資助比例如下：

    　　　資助比例

    　　　450元或以下：100%全數資助

    　　　451-1,200元：薪酬 X 50%；以450元為資助起點，例如：薪酬為500元 X 50% ＝250元，亦可獲450元資助〔註：運輸從業員的薪酬，會以每台的總金額，來決定是否使用$450資助起點〕


    B)　因防疫而產生支出的定額資助$2,000，用途包括但不限於：

    　—　因防疫需要而採購的用品；

    　—　因防疫需要的支出，如：增加人手等


    C)　參考例子：一場聘用72名工作人員的演出，$18,000的資助，可支持39位基層從業員的酬金 （資助額計算見7A點）：


    配戲演員：$700 X 1名 = $450、$600 X 1名 = $450、$500 X 2名 = $900

    下欄演員：$420 X 9名 = $3,780

    武師：$1000 X 1名 = $500、$800 X 5名 = $2,250

    音樂：$1000 X 7名 = $3,500

    畫部工人：$700 X 6名 = $2,700

    服裝：$700 X 1名 = $450、$650 X 3名 = $1,350、$600 X 3名 = $1,350

    （甲）39名從業員：合共$17,680 + （乙）防疫支出：$2,000＝總資助額 $19,680 


    **申請方法**

    9.　本計劃於2021年7月22日開始接受報名，截止申請日期為2021年8月4日。必須由劇團的營運者向香港八和會館提出申請，如有需要，營運者亦可以書面授權形式，授權予其所指定的人士處理申請手續，但申請表格必須由營運者親自簽署。此外，申請人必需填妥由八和提供的申請表格，營運者需要簽名及蓋章（劇團印鑑）作實，否則將不獲處理


    10.　在提出申請時，申請團體必須附上申請演出的**場地證明文件、營運者之身份證副本**，否則將不獲處理


    **發放資助**

    11.　本計劃預計於10月（確實日子請留意本會的最新公告）公布申請結果


    12.　劇團的營運者須先行墊支獲資助的款額，本計劃將於演出後才發放資助


    13.　演出完成以後，劇團營運者須提交由每位乎合資格之基層從業員個別簽收的酬金証明文件副本、及因防疫而產生支出的定額資助$2,000的實際用途之書面說明。每位乎合資格之基層從業員個別簽收的酬金証明文件副本上之酬金需與申請表格上填寫該從業員的酬金一致，並且每位每場演出之酬金不可多於$1,200


    14.　獲資助的劇團於演出完成後，須於8月25日前向香港八和會館提交第13點列出的資料，逾期提交者，資助將不獲處理


    15.　待香港八和會館核實資料後，將通知獲資助團體的負責人到會館領取資助，資助將以支票形式發放，支票抬頭將以申請表上之「劇團名稱」填寫


    **查詢及資助公布**

    16.　如就本計劃有任何查詢，歡迎聯絡八和職員周朗峰，電話：2384-2929、電郵：david.chow@hkbarwo.com


    17.　香港八和會館將就本計劃將不時發放最新資料，如欲獲取最新資訊，請留意本會網址：www.hkbarwo.com

    2.　營運者之身份證副本

    3.　已填妥及具備簽名、蓋章之申請表格


    8月25日前

    5.　因防疫而產生支出的定額資助$2,000的實際用途之書面說明
